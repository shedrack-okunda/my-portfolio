## ğŸš€ Getting Started

### Prerequisites

-   Node.js (v18 or higher)
-   npm or yarn
-   Git

### Installation

1. **Clone the repository**

    ```bash
    git clone https://github.com/yourusername/portfolio.git
    cd portfolio
    ```

2. **Install dependencies**

    ```bash
    npm install
    # or
    yarn install
    ```

3. **Start the development server**

    ```bash
    npm run dev
    # or
    yarn dev
    ```

4. **Open your browser**
   Navigate to `http://localhost:5173`
